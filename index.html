O que é Dart

Dart é uma linguagem de programação moderna, criada pelo Google, projetada para ser fácil de aprender e poderosa o suficiente para criar aplicações complexas.

Características Principais

  • Orientada a objetos: Tudo em Dart é um objeto
  • Null-safety: Sistema que previne erros comuns
  • Compilada e interpretada: Alta performance com ótimo desenvolvimento
  • Sintaxe familiar: Similar a Java, C# e JavaScript

História Breve da Linguagem

Dart nasceu em 2011 dentro do Google, criada por uma equipe liderada por Lars Bak e Kasper Lund.

Marcos importantes:

  • 2011: Anúncio público do Dart
  • 2013: Primeira versão estável (1.0)
  • 2018: Dart 2.0 com sistema de tipos robusto
  • 2021: Dart 2.12 com null-safety obrigatório
  • 2023: Dart 3.0 com records e patterns
